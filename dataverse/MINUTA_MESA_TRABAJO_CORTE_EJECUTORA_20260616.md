## Propósito

Esta es la hoja comun de conversacion previa para la corte ejecutora.

La orden viene de Dataverse, se habla entre todos, se escucha el delta y despues se prepara el acta.

## Mesa

1. Hablar primero.
2. Leer la orden viva entre todos.
3. Acordar el delta.
4. Definir quien redacta el acta.
5. Cerrar evidencia.
6. Dejar el siguiente movimiento listo.

## Formato De Respuesta

- orden recibida
- agente responsable
- siguiente paso
- evidencia
- stop condition

## Criterio

No se inventa una activacion nueva.
La mesa solo conversa, ordena lo que ya existe y deja listo el acta para continuar.
