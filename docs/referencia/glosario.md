# Glosario

| Termino | Definicion inicial |
|---|---|
| SDU | Sistema Declarativo Universal. |
| Corte | Conjunto de agentes institucionales que revisan, ordenan y ejecutan bajo gates. |
| Wave | Pasada atomica de trabajo con objetivo, evidencia y cierre. |
| Huella | Evidencia atomica que deja continuidad para el siguiente agente o proceso. |
| Seshat | Agente/capa de canon documental y memoria institucional. |
| Readback | Cierre escrito que devuelve estado, evidencia, decision y siguiente delta. |
| Gate | Condicion explicita de avance, validacion o decision humana. |
| Dev Drive | Volumen optimizado para desarrollo en Windows. |
