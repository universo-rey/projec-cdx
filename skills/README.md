# Skills

Capacidades reutilizables de la cabina `PROJEC CDX`.

La cobertura visible se lee en [docs/superpowers/plans/README.md](C:/Users/enzo1/PROJEC%20CDX/docs/superpowers/plans/README.md) y en el plan rector de cobertura total.

## Uso

- Una skill encapsula una capacidad estable.
- Si cambia una skill, se sincroniza la cadena.
- `matrix-recipe-skill-sync` es la referencia para alinear recetas y skills.

## Regla

- No dejar conocimiento operativo solo en conversación.
- Promover aprendizajes estables a skill, receta, tool o índice.
