# Start Here Cierre Wave

Entrada corta para waves de cierre, archivo o absorcion de hitos.

## Orden

1. Leer `operativa/CURRENT.md`.
2. Leer `operativa/TRACE.md`.
3. Leer `operativa/NEXT.md`.
4. Leer `recipes/cierre-wave-documental.md`.
5. Leer `hitos/INDICE_MAESTRO.md`.
6. Leer `operativa/MATRIZ_SKILLS_TOOLS_RECETAS_20260615.md`.
7. Leer `operativa/PROMPT_CIERRE_WAVE.md` si hace falta reanudar el hilo.
8. Si el frente toca una superficie concreta, leer `operativa/ANCLAS_ON_DEMAND.md` y abrir solo la ancla minima que corresponda.

## Regla

- Un solo delta.
- Un solo archivo o hito a la vez.
- No repetir patrones: aplicar la receta.
- Dataverse queda on-demand, no fijo en la ruta principal.

## Flujo

1. Identificar el delta nuevo real.
2. Decidir si genera archivo, matriz o hito.
3. Reflejar solo las superficies tocadas.
4. Versionar si es durable.
5. Validar con `tools/validate_proj_cdx_workbench.ps1`.

## Stop Condition

- Falta evidencia.
- Falta validador.
- Falta ruta visible.
- Se intenta abrir un carril live por inferencia.
