# Patron - Seshat Home Canonico

## Uso

Usar cuando una huella atomica aprobada deba quedar visible en el sitio
`SeshatHubRegistroN.8` como lectura canonica para agentes.

## Regla

- `Home.aspx` es la superficie primaria visible.
- Si la API de paginas no permite editar, publicar un artefacto canonico en el
  sitio y registrar `delta_gobernado` para el binding de pagina.
- Dataverse debe conservar puntero metadata-only.
- La lectura inicia con identidad, pertenencia, Corte y proposito.

## Evidencia Minima

- URL del sitio.
- URL del archivo o pagina.
- Item id si existe archivo.
- Canonical id Dataverse.
- Rollback y postcheck.
