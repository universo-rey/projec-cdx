# Perfil Windows - Snapshots y GMT

## Regla

Tratar `@GMT` como instantanea historica, no como carpeta activa. Resolver fecha, origen y alcance antes de abrir o modificar.

## Uso

Aplicar cuando un path contenga `@GMT` o una ruta restaurada parezca superficie viva.

## Salida

Instantanea reconocida como historial y superficie activa separada con claridad.
