# Workbench Atomico Accionable

## Regla

Cada entrada visible debe ser breve, clara y empujar una accion.

## Uso

Aplicar cuando una raiz, mapa o README empieza a crecer y pierde energia operativa.

## Fuente

- `README.md`
- `atomic/README.md`

## Salida

- Entrada corta.
- Detalle movido a indice, referencia, patron o proceso.
- Siguiente paso explicito.
